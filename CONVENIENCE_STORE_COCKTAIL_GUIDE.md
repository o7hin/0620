# 🏪 便利商店調酒系統使用指南

## 📋 功能概述

本系統專門設計用於生成**便利商店調酒**配方，讓您可以使用台灣各大便利商店（7-11、全家、萊爾富、OK超商）購買的現成材料，在家輕鬆製作美味調酒。

## 🍸 主要特色

### ✅ 材料限制
- **只使用便利商店可購買的商品**
- 避免專業調酒材料和器具
- 成本控制在 100-200 元之間
- 適合在家製作

### 🏪 支援的便利商店品牌
- **7-11** (統一超商)
- **全家** (FamilyMart)
- **萊爾富** (Hi-Life)
- **OK超商** (OK mart)

## 🛒 常見材料清單

### 🍺 酒類商品
- 台灣啤酒、海尼根啤酒、Corona啤酒
- 梅酒、柚子酒、水蜜桃酒
- 燒酎、威士忌、伏特加、白蘭地

- 養樂多、優酪乳、綠茶、咖啡

### 🍋 配料裝飾

## 🎯 使用方法

### 1. 開啟調酒功能
1. 在網頁中點擊「🏪 便利商店調酒」標籤
2. 系統會自動生成一款便利商店調酒配方

### 2. Google API 生成（推薦）
1. 設定 Google API Key 獲得更個性化的配方
2. 系統會根據便利商店材料生成創意調酒
3. 包含詳細的購買建議和製作步驟

### 3. 模擬數據（無需 API）
1. 未設定 API Key 時使用預設配方
2. 包含多種不同酒精濃度的選擇
3. 隨機選擇不同的便利商店調酒

## 💡 調酒範例

### 🥃 經典款：7-11 威士忌可樂
```
材料：
- 威士忌 30ml（便利商店小瓶裝）
- 可口可樂 150ml
- 冰塊 適量
- 檸檬片 1片

製作步驟：
1. 在玻璃杯中放入冰塊
2. 倒入威士忌
3. 緩慢倒入可樂，避免泡沫過多
4. 用檸檬片裝飾杯緣
5. 輕輕攪拌 2-3 下即可享用

風格：經典便利商店風格 🥃 | 中等酒精濃度
```

### 🍋 清爽款：全家梅酒氣泡
```
材料：
- 梅酒 40ml（便利商店常見品牌）
- 檸檬汽水 120ml
- 薄荷葉 2-3片
- 冰塊 適量

製作步驟：
1. 在杯中加入冰塊
2. 倒入梅酒
3. 加入檸檬汽水
4. 放入薄荷葉增添香氣
5. 用吸管輕輕攪拌

風格：清爽便利商店風格 🍋 | 低酒精濃度
```

## 🛡️ 注意事項

### 飲酒安全
- 請適量飲酒，切勿過量
- 未滿 18 歲請勿飲酒
- 酒後請勿開車

### 材料替代
- 如某些材料缺貨，可尋找類似商品替代
- 建議先購買基本材料（酒類、飲料、冰塊）
- 裝飾材料可依個人喜好調整

### 製作技巧
- 冰塊建議使用大塊冰，融化較慢
- 氣泡飲料要緩慢倒入，避免泡沫過多
- 可依個人口味調整材料比例

## 📞 故障排除

### 調酒生成失敗
1. 檢查網路連線
2. 確認 Google API Key 是否正確設定
3. 嘗試重新整理頁面

### 材料找不到
1. 嘗試不同的便利商店品牌
2. 詢問店員是否有類似商品
3. 使用系統建議的替代材料

### API 相關問題
1. 檢查 API Key 是否有效
2. 確認 API 額度是否足夠
3. 聯繫 Google AI 支援
